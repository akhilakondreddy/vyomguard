VyomGarud Landing Page

A modern, responsive landing page for VyomGarud, a UAV / drone systems company, built with React and Tailwind CSS. Designed to be sleek, dark-themed, and professional with a military/futuristic vibe.

Features

Hero Section with company name, tagline, CTA, and bold visuals.

About Section with a short summary of VyomGarud’s mission.

Capabilities Section showcasing 3-4 product cards.

Highlights Section with 3 concise feature bullets.

Contact Section with a simple contact form.

Smooth horizontal scrolling and subtle animations using Framer Motion.

Responsive design for desktop and mobile.

Tech Stack

React (Frontend library)

Tailwind CSS (Styling)

Framer Motion (Animations)

Node.js & npm (Development tools)

Getting Started
1. Clone the repository
git clone <your-repo-link>
cd vyomgarud

2. Install dependencies
npm install

3. Start the development server
npm start


Opens the site at http://localhost:3000.

4. Build for production
npm run build
